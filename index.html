Was ist JavaScript?

JavaScript ist eine Programmiersprache, die Entwickler verwenden, um interaktive Webseiten zu erstellen. Von der Aktualisierung von Social Media Feeds bis hin zur Anzeige von Animationen und interaktiven Karten können JavaScript-Funktionen die Benutzerfreundlichkeit einer Website verbessern. Als clientseitige Skriptsprache ist sie eine der Kerntechnologien des World Wide Web. Wenn Sie zum Beispiel im Internet surfen und ein Bildkarussell, ein Dropdown-Menü zum Anklicken oder dynamisch wechselnde Elementfarben auf einer Webseite sehen, sehen Sie die Auswirkungen von JavaScript.
Quelle: https://aws.amazon.com/de/what-is/javascript/ (Abschnitt "Was ist Javascript")

Geschichte von JavaScript

JavaScript wurde 1995 von Brendan Eich für den Netscape Navigator entwickelt. Die Sprache entstand in einer Zeit, in der das Web noch in den Kinderschuhen steckte und Interaktivität dringend benötigt wurde. Mit der Einführung von Technologien wie AJAX und modernen Frameworks hat JavaScript die Art und Weise, wie wir das Web erleben, revolutioniert.
Quelle: https://de.wikipedia.org/wiki/JavaScript (Abschnitt "Geschichte" zusammengefasst)

Warum wurde JavaScript erfunden?

JavaScript (kurz JS) ist eine Skriptsprache, die ursprünglich 1995 von Netscape für dynamisches HTML in Webbrowsern entwickelt wurde, um Benutzerinteraktionen auszuwerten, Inhalte zu verändern, nachzuladen oder zu generieren und so die Möglichkeiten von HTML zu erweitern. Heute wird JavaScript auch außerhalb von Browsern angewendet, etwa auf Servern und in Microcontrollern.
Quelle: https://de.wikipedia.org/wiki/JavaScript (Abschnitt "Javascript")

Vorteile und Nachteile von JavaScript

Vorteile

  • Einfach zu lernen
  • Plattformunabhängig
  • Viele verschiedene Einsatzmöglichkeiten

Nachteile

  • Manchmal inkonsistentes Verhalten
  • Kann zu Sicherheitsrisiken führen (z.B. XSS-Attacke)
  • Debugging kann schwierig sein

Kurzer Info-Text zu TypeScript

TypeScript ist ein Superset von JavaScript, das JavaScript um optionale statische Typisierung und erweiterte Funktionen erweitert. Es wurde von Microsoft entwickelt und erstmals im Oktober 2012 veröffentlicht. Seit seiner Veröffentlichung im Jahr 2012 hat es sich in der Webentwicklergemeinschaft schnell durchgesetzt.
